Stretch Challenge

In future weeks you will have tasks where you are required to write your own unit tests for happy paths and edge cases that your team comes up with. In preparation for this, we encourage you to complete this stretch challenge.

Write additional unit tests for bst_tasks:contains_test_/0 covering the case where student descriptors, {name,age,standing}, are in the bst. Specifically include the edge cases:

  1. a nil bst being searched to see if it contains a specified student description,
  2. a non-nil bst being searched to see if it contains a specifed student description that it does not contain, and
  3. any other edge cases you can think of.

Write additional unit tests for rbt_tasks:contains_test_/0 covering the case where student descriptors, {name,age,standing}, are in the bst. Add as many happy paths and edge cases as are appropriate.

Review

After completing the previous tasks:

  1. Review one of the many possible solutions for this task,
  2. Compare and contrast your solutions with the possible solutions, and
  3. Make note of any improvements you could make for future tasks.
