CSE 382: Week 05

Ponder and Prove

Task 1

Your tasks are included in the tasks.erl and speed.erl files found in the chain_curry_partial zip file.

Task 1

Complete the implementation of the chain function in chain_curry_partial/src/tasks.erl so that it passes all the unit tests in chain_curry_partial/src/tasks.erl.

Task 2

Complete the implementation of the curry function in chain_curry_partial/src/tasks.erl so that it passes all the unit tests in chain_curry_partial/src/tasks.erl.

Task 3

Find a sophisticated example of code in your language that is part of an opensource library. Examine it with regard to the definition of beautiful code in the reading. In what ways is the code beautiful and in what ways is it not? Describe in writing how would you change the code to enhance its beauty.

Task 4

Find some code you've written in the past that could be improved by applying chaining. Consider why it would be an improvement. Write rules you come up with regarding when chaining is appropriate and when it is not.

Review

After completing the previous tasks:

  1. Review one of the many possible solutions for this task,
  2. Compare and contrast your solutions with the possible solutions, and
  3. Make note of any improvements you could make for future tasks.
