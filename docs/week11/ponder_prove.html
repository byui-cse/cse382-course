Your Tasks

Task 1

Write code implementing the trie_node, \(lookup\), and \(add\) pseudocode from the reading and write code that exercises the functions you've written.

Task 2

Write pseudocode and code in your language that implements a tries monad. Include all the required functions, based on the rules for what is and is not a monad. Also write a binding function for your tries monad.

Task 3

Gather data from at least 20 people. This should include the country, state, city, and street address along with their last and first names. Create strings from this data where the string is \(country\doubleplus state\doubleplus city\doubleplus street\_address\doubleplus last\_name\doubleplus first\_name\). Draw a picture, similar to the images in the reading, of what this data would look like in a trie. Exercise your code by adding this data and doing multiple searches for the each person's complete data.

Review

After completing the previous tasks:

  1. Review one of the many possible solutions for this task,
  2. Compare and contrast your solutions with the possible solutions, and
  3. Make note of any improvements you could make for future tasks.
