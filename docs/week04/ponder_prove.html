CSE 382: Week 04

Tasks

Your Tasks

Task 1

Write and test a custom implementation of the foldl functor called c_foldl (see Figure 2 for the description).

Task 2

Use c_foldl to create a single string from a list of 200 strings, words of your choice.

Task 3

Use the appropriate fold BIF from your language to build the same piece of text. Compare the speed and the accuracy of the two implementations.

Task 4

Write and test a custom implementation of the foldr functor called c_foldr (see Figure 3 for the description).

Task 5

Use c_foldr to create a single string from the same list of 200 strings, words of your choice.

Task 6

Use the appropriate fold BIF from your language to build the same piece of text. Compare the speed and the accuracy of the two implementations.

Task 7

Write and test a custom implementation of the unfold functor called c_unfold (see Figure 4 for the description).

Task 8

After completing Tasks 1 through 7, find the source code for the BIFs for your chosen language and compare them to your implementations of the foldl and foldr functors. Speculate, in writing, as to why the standard implementations differ, or not, from yours.
