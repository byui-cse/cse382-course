Ponder and Prove

Your Tasks

Task 1

Create pseudocode for and, using your language, write as a monoid the following set operations.

  1. union
  2. intersection
  3. difference
  4. symmetric_difference
Write tests for your code.

Task 2

Consider these functions that work with sets. Are they part of the monoid from Task 1? Why or why not?

  • is_element_of
  • create_subset_where
  • is_subset_of
  • Assuming the elements of the set are numeric, a pair of functions called divide_all_by and add_to_all

    • Task 3

    Consider a monad that is the set of functions that build http GET and POST requests.

    1. In your language, write the \(bind\) and \(unit\) functions for this monad.
    2. Write the pseudocode and code in your language for the functions for this monad that creates a tuple that includes the base URL, the directory extensions, and whether the request is a GET or POST request.
    3. Write the pseudocode and code in your language for the function that builds a tuple representing a POST request with POST data.
    4. Write tests for your code.
