Your Tasks

Task 1

Because lists are such a fundamental part of every functional programming language, it would be a waste of your time to ask you to implement a list from scratch in your language. Instead you will be drawing sketches of what happens to lists as they are manipulated. Look forward to writing code in next week.👍🎉😎

For each of these tasks below, draw, step-by-step, how the list is created, accessed, or modified.

  1. Start with a list with one element. Prepend 5 other items to the list.
  2. Start with a list with one element. Append 5 other items to the list.
  3. Accessing the Head and Tail of a List
  4. Inserting into a List

Task 2

For your language, find the BIFs and their big-O complexities for each of the behaviors listed above. The big-O complexity of each BIF may be listed in your language's official documentation, or you may need to do web searches to find it. List each function, its big-O, and speculate on how the BIF may be written to achieve that big-O complexity.
